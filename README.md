# ApexPlanet Web Development Task 3

## Project Overview

This project was developed as part of Task 3 of my Web Development Internship at ApexPlanet Software Pvt Ltd.

## Features

* Responsive Web Design using Media Queries
* Interactive Image Carousel using JavaScript
* API Integration using Fetch API
* Mobile, Tablet, and Desktop Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Functionality

### Image Carousel

Users can navigate through multiple images using Previous and Next buttons.

### API Integration

The application fetches data from an external API and displays it dynamically on the webpage.

### Responsive Design

The webpage adapts to different screen sizes for a better user experience.
